Directives

The simplest way to think about a directive is that it is simply a function that we run on a particular DOM element. The directive function can provide extra functionality on the element.

Built in Directives

  • ng-href
  • ng-src
  • ng-disabled
  • ng-checked
  • ng-readonly
  • ng-selected
  • ng-class
  • ng-style

Custom Directives

A directive is defined using the .directive() method, one of the many methods available on our applicationâ€™s Angular module.

The myDirective directive definition looks like:


angular.module('myApp', [])
directive('myDirective', function() {
return {
restrict: 'E',
template: '
Click me to go to Google'
}
});

Invoking the directive

  • element (E)
  • an attribute (A)
  • a class (C)
  • a comment (M)


For example 


  restrict: 'EAC'
