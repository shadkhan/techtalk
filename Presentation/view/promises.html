promise

Promises are a way of registering interest in something that will happen  asynchronously, such as the response sent  from a server for an Ajax request. 
  • A promise is an object with a"then"   method
  •  promise.then(onSuccess, onFailiure);

onSuccess function is called whenever the asynchronous tasks ends successfully and the onFailure function is called if the tasks fails to execute

$q

  • In Angular $q service  provide deferred and promise implementations.
  •  We can get a new deferred object by calling the defer() function on the $q service.
  • the deferred object has two methods 
    • resolve(…) - used to signal that the task has succeeded
    • reject(…) - used to signal that the task has failed
