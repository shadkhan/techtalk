$scope

  • Provides context for expression evaluation
  • Behave same as  DOM`s  hierarchy structure
  • Fallows prototypical inheritance

  • request $scope in controllers
  myApp.controller('GreetCtrl', function ($scope) {
    $scope.greeter = { greeting: 'Misko'};
});
  • Use in view/template 
  <p ng-controller='GreetCtrl'>{{greeter.greeting}}</p>
