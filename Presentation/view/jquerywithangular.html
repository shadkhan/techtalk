ANGULAR WITH JQUERY

  • Angular uses jQuery if it's presentwhen the application is being bootstrapped.
  • If it is not present , Angular falls back to its own implementation called  jQLite.
  • jQLite is a tiny subset of jQuery that allows Angular to manipulate the DOM in a cross-browser compatible way.with the goal of having a very small footprint.


Using JQUERY PLUGIN

For using a jQuery plugin, do not put the code in the controller. Instead create a directive anut the code that you would normally have inside the link function of the directive.. So what we need to do is
  • include jQuery in your HTML
  • call initialization function in directive link function





Example

Creating a directive with Toolbar.js jQuery plugin
 <div id="format-toolbar1" class="settings-button" toolbar-tip="{content: '#format-toolbar-options', position: 'top'}">
    <img src="http://paulkinzett.github.com/toolbar/img/icon-cog-small.png">
</div>
 var App = angular.module('Toolbar', []); 
App.directive('toolbarTip', function() {
    return {
        restrict: 'A',
        // responsible for registering DOM listeners as well as updating the DOM
        link: function(scope, element, attrs) {
            $(element).toolbar(scope.$eval(attrs.toolbarTip)); 
//;Rather than using $(element), you can make use of angular.element(element) when using AngularJS with jQuery
        }
    };
});
